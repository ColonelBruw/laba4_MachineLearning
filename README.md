# Распознование лиц на основе моделей SSD и VVGFace
## *Выполнил: Чугунков В. С.*

## :information_source: Цель: 
- Написать детекции для предобученных моделей распознавания лиц: **ssd300_vgg16** из библиотеки torchvision и **VGGFace**
- Сравнить **эффективность работы моделей**, вычислив евклидово расстояние между векторами признаков изображений

## :bookmark_tabs: Задачи:
1. Загрузить предобученнную модель **ssd300_vgg16** из библиотеки torchvision
2. Загрузить **несколько изображений** лиц и конвертировать их в тензоры
3. Написать **функции для детекции изображений и отрисовки bounding box'ов** обнаруженых объектов
4. **Протестировать распознавание лица**, загрузив в функцию детекции изображение. Область лица на изображении должна быть обведена квадратом с подписью person
5. Найти **евклидово расстояние** между веторами признаков изображений и определить точность детекции. Чем выше значение расстояния, тем больше сходства между фото
6. Вышеописаные шаги провести с моделью **VGGFace**

## :white_check_mark: Итоги
В ходе выполнения работы были протестированы модели **ssd300_vgg16 и VGGFace**, а также написаны функции предобработки и детекции для них
- Обе модели **стабильно распознают лица** на фото, при условии **отсутствия посторонних объектов** и **удовлетворительного качества**
